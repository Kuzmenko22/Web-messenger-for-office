# Проект на T3 Stack

## Установка
1) Клонировать репозиторий

   ``git clone https://github.com/Kuzmenko22/Planirovanie-zadach.git``
   
   ``cd Planirovanie-zadach``

3) Установить зависимости
   
   ``pnpm install``

5) Инициализация БД
   
   ``pnpm prisma migrate dev --name init``

7) Запуск контейнеров
   
   ``pnpm db:start``
   
9) Запуск разработки
    
   ``pnpm dev``
