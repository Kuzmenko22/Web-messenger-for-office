# Проект на T3 Stack

## Установка
1) Клонировать репозиторий

   ``git clone https://github.com/Kuzmenko22/Web-messenger-for-office.git``
   
   ``cd Web-messenger-for-office``

3) Установить зависимости
   
   ``pnpm install``

5) Инициализация БД
   
   ``pnpm prisma migrate dev --name init``

7) Запуск контейнеров
   
   ``pnpm db:start``
   
9) Запуск разработки
    
   ``pnpm dev``
